# GOSH Learning Project

GOSH is a shell written in Go, featuring the following:

## Features

1. Configurable in Go
   - Needs a Go interpreter
2. Command/output as data
   - Queryable based on timestamp, exit code
   - Snapshot and restore shell state
3. POSIX compatible

## Purpose

The primary purpose of this project is for me to learn more about operating system. I want to know how to control processes and understand various aspects of a shell.
